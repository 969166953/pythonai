# 更新日志

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)，版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2026-04-15

### 新增
- 知识库创建、编辑、删除
- 文档上传处理（PDF、DOCX、DOC、TXT、Markdown）
- RAG 流式对话，支持多轮上下文
- 回答来源引用，点击展开完整内容
- 文档分块预览
- 暗色/亮色主题切换，跟随系统偏好
- 对话重命名和删除
- 代码块语法高亮和一键复制
- Toast 全局通知
- 删除操作确认弹窗
- 错误边界防止页面白屏
- 响应式布局（桌面侧栏 + 移动端抽屉）
- Docker Compose 一键部署

### 安全
- 文件上传大小限制 (50MB)
- API 输入长度校验
- 对话归属校验，防止跨知识库访问
- LLM 错误信息脱敏
- 安全响应头 (X-Frame-Options, X-Content-Type-Options)
- ChromaDB 操作异步化，不阻塞事件循环
